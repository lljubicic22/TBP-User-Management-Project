# TBP-User-Management-Project

Tema projekta je upravljanje korisničkim računima, ulogama i dozvolama uz primjenu role-based access control (RBAC) modela.

Aplikacija omogućuje:

* autentifikaciju korisnika
* upravljanje korisnicima, ulogama i dozvolama
* prilagodbu grafičkog sučelja ovisno o ulozi korisnika

Sustav razlikuje administratora, menadžera i običnog korisnika, pri čemu svaka uloga ima jasno definirana prava pristupa.

Projekt je izrađen u sklopu kolegija Teorija baza podataka i temelji se na PostgreSQL objektno-relacijskoj bazi podataka, Flask backendu i HTML/CSS/JavaScript grafičkom sučelju.
